# Mystic Mist Watch App UI Design

## Overview

Mystic Mist Watch is a smartwatch companion application designed to provide quick access to perfume products and user account features.

The design focuses on usability, accessibility, and navigation within limited screen space.

## Features

- Product Browsing
- User Account Access
- Smartwatch-Friendly Navigation
- Minimal User Interface

## Tools Used

- Figma
- UI Design
- UX Design
- Prototyping
